# gatilho-yt

Gatilho HTTP para Burp — dar view no vídeo fixo (sem player).

**Vídeo:** `https://www.youtube.com/watch?v=2sTS91HNoIs` (início **0s**, sem `t=41`)

Config em `video.config.js`.

- `/` ou `/v` — dispara view
- `/api/view` — gatilho Burp (sem parâmetros)

**Burp:** `GET /api/view` ou `GET /v`
